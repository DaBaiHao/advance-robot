# How to run 
---

### 1. The first thing need to open the TERMINAL
### 2. Type `roscore` in TERMINAL(don't forget `catkin_make`)
### 3. Open another TERMINAL, Type `roslaunch AR_week5_test cubic_traj_gen.launch`
### 4. Open another TERMINAL, Type `rqt_plot`
### 5. In the rqt_plot window, type `/plot` to see the figure
